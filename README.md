# 🛸 Mesh

The terminal AI coding agent that stays within context.

```bash
npm i -g @edgarelmo/mesh-agent-cli
mesh
```

- **Zero Config**: Works out of the box. No AWS keys needed.
- **Speech-to-Speech**: Type `/voice` to talk directly with your agent (local STT/TTS).
- **Context Compression**: Keeps huge codebases in-window.
- **Smart UX**: Slash commands, TAB completion, and ghost text.
- **Persistent**: Remembers your history and workspace intelligence.

## ❯ commands
- `/voice` - Toggle Speech-to-Speech mode
- `/setup` - Configuration wizard
- `/model` - Switch models (Sonnet 4.6 default)
- `/index` - Optimize workspace indexing
- `/status` - Check health & token usage
